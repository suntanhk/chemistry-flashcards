3.1
Electron Loss & Gain
▾

Modern definitions (electron transfer)

A redox reaction is a reaction that involves both reduction and oxidation happening at the same time.

  • Oxidation is the loss of electrons.
  • Reduction is the gain of electrons.

Remember using the mnemonic: OIL RIG — Oxidation Is Loss, Reduction Is Gain (of electrons).

Older definitions (oxidation = gain of oxygen/loss of hydrogen; reduction = loss of oxygen/gain of hydrogen) still work for simple cases, but the electron-transfer definition is the one you must use and quote at A level, since it also applies to reactions with no oxygen or hydrogen involved at all.

Oxidising agents and reducing agents

  • Oxidising agent — a species that oxidises another species by removing electrons from it. In doing so, the oxidising agent itself gains electrons and is therefore reduced.
  • Reducing agent — a species that reduces another species by adding electrons to it. In doing so, the reducing agent itself loses electrons and is therefore oxidised.
Classic trap: "oxidising agents gain electrons" and "reducing agents lose electrons" — this describes what happens to the agent itself, which is the opposite of what it does to the other species. Read the question carefully to see which species you're being asked about.

Disproportionation — a first look

Disproportionation is a special type of redox reaction where the same element, in the same starting species, is simultaneously oxidised and reduced.

Example: 2Cu⁺(aq) → Cu²⁺(aq) + Cu(s) — one Cu⁺ ion loses an electron (oxidised to Cu²⁺) while another Cu⁺ ion gains an electron (reduced to Cu). Both changes happen to the same element, copper, in the same reaction.

Definition
State the electron-transfer definitions of oxidation and reduction.
Oxidation is the loss of electrons. Reduction is the gain of electrons. (OIL RIG)
Careful!
What happens to an oxidising agent itself during a reaction?
It gains electrons and is therefore reduced (even though it oxidises the other species).
Definition
Define disproportionation.
A redox reaction in which the same element, in the same species, is simultaneously oxidised and reduced.
Apply
In Zn(s) + CuSO₄(aq) → ZnSO₄(aq) + Cu(s), which species is the reducing agent?
Zinc — it loses electrons (is oxidised) and in doing so reduces the Cu²⁺ ions, so Zn is the reducing agent.
3.2
Oxidation Numbers
▾

What is oxidation number?

Oxidation number is the charge that an atom/ion has, or would have if the species were fully ionic. It lets us track electron transfer even in covalent compounds, where a simple ionic picture doesn't apply.

Rules for determining oxidation number

  • The oxidation number of an uncombined element is 0 (e.g. Cl₂, Fe, O₂).
  • The oxidation numbers of all elements in a neutral compound sum to 0.
  • The oxidation numbers of all elements in an ion sum to the charge on that ion.
  • The more electronegative element in a substance is given the negative oxidation number.
  • Fluorine is always −1.
  • Hydrogen is +1, except when bonded to a less electronegative element (e.g. metal hydrides like NaH), where it is −1.
  • Oxygen is −2, except in peroxides (e.g. H₂O₂) where it is −1, and when combined with fluorine, where it is positive.

Worked examples

Chlorine in NaClO₃: Na is +1, O is −2 (×3 = −6). Sum must be 0, so Cl = 0 − 1 − (−6) = +5.

+1 + Cl + (3 × −2) = 0 → Cl = +5

Nitrogen in NO₂⁻: O is −2 (×2 = −4). Sum must equal the ion charge, −1, so N = −1 − (−4) = +3.

N + (2 × −2) = −1 → N = +3

Method: assign the elements you already know (usually O and H first, using the exceptions list), then use the "sum = 0 (compound) or sum = charge (ion)" rule to solve for the unknown element.
Definition
Define oxidation number.
The charge an atom/ion has, or would have if the species were fully ionic.
Calculate
What is the oxidation number of Cr in Cr₂O₇²⁻?
O is −2 (×7 = −14). Sum = −2 (ion charge). 2Cr + (−14) = −2, so 2Cr = +12, Cr = +6.
Exception
What is the oxidation number of oxygen in H₂O₂, and why is it different from usual?
−1. H₂O₂ is a peroxide, and oxygen is −1 (not the usual −2) in peroxides.
Exception
What is the oxidation number of hydrogen in NaH, and why?
−1. Hydrogen is bonded to sodium, which is less electronegative than hydrogen, so H takes the negative oxidation number here (reversing the usual +1).
3.3
Using Oxidation Numbers to Classify Reactions
▾

Redox definition using oxidation numbers

  • An element is oxidised when its oxidation number increases.
  • An element is reduced when its oxidation number decreases.

This gives a quick, general way to spot a redox reaction: work out the oxidation number of every element on both sides of the equation. If any element's oxidation number changes, the reaction is redox.

Worked example — is this reaction redox?

H₂S(g) + Cl₂(g) → 2HCl(g) + S(s)

S goes from −2 (in H₂S) to 0 (uncombined) — oxidation number increases, so sulfur is oxidised.

Cl goes from 0 (in Cl₂) to −1 (in HCl) — oxidation number decreases, so chlorine is reduced.

Both oxidation and reduction occur → this is a redox reaction.

By contrast, NaOH(aq) + HCl(aq) → NaCl(aq) + H₂O(l) has no oxidation number changes for any element — it is an acid-base neutralisation, not a redox reaction.

Spotting disproportionation from oxidation numbers

2NaOH(aq) + Cl₂(aq) → NaCl(aq) + NaClO(aq) + H₂O(l)

Chlorine starts at 0 in Cl₂. In the products, it is −1 in NaCl (reduced) and +1 in NaClO (oxidised) — the same element, chlorine, has been both oxidised and reduced in the same reaction. This is disproportionation.

Rule
In terms of oxidation number, when is an element oxidised?
When its oxidation number increases.
Classify
Is CaO + H₂O → Ca(OH)₂ a redox reaction? Explain.
No — Ca stays +2, O stays −2, H stays +1 throughout. No oxidation numbers change, so it is not a redox reaction.
Classify
Is 2H₂O₂ → 2H₂O + O₂ a disproportionation reaction?
Yes — oxygen goes from −1 in H₂O₂ to −2 in H₂O (reduced) and to 0 in O₂ (oxidised). The same element (oxygen, from the same compound) is both oxidised and reduced.
3.4
Systematic Naming & Writing Formulae
▾

Systematic (Roman numeral) names

When an element can exist in more than one oxidation state, its oxidation number is shown as a Roman numeral in brackets straight after the element's name.

FormulaOxidation numberSystematic name
FeCl₂Fe = +2iron(II) chloride
KMnO₄Mn = +7potassium manganate(VII)
K₂MnO₄Mn = +6potassium manganate(VI)
Na₂SO₄S = +6sodium sulfate(VI)
Na₂SO₃S = +4sodium sulfate(IV)
You don't need a Roman numeral if the element only ever has one oxidation state in its compounds — e.g. "sodium chloride" (Na is always +1) not "sodium(I) chloride", and "magnesium oxide" not "magnesium(II) oxide".

Working backwards: formula from oxidation number/ion charge

Example — deduce the formula for iron(III) sulfate:

  • Iron(III) ion = Fe³⁺
  • Sulfate ion = SO₄²⁻
  • Combine in a ratio that balances charge: 2 Fe³⁺ (+6 total) with 3 SO₄²⁻ (−6 total)
  • Formula: Fe₂(SO₄)₃
Naming
Give the systematic name for KMnO₄.
Potassium manganate(VII) — Mn has oxidation number +7.
Formula
Deduce the formula for chromium(III) sulfate(VI).
Cr³⁺ and SO₄²⁻ combine in a 2:3 ratio to balance charge: Cr₂(SO₄)₃.
Explain
Why don't we need to say "sodium(I) chloride"?
Sodium only ever exists as +1 in its compounds, so there's no ambiguity to resolve — the Roman numeral is only needed when an element has more than one possible oxidation state.
3.5
Half-Equations & Constructing Full Redox Equations
▾

Straightforward case — electrons already match

If both half-equations contain the same number of electrons, just add them together and cancel the electrons:

Zn(s) → Zn²⁺(aq) + 2e⁻ Cu²⁺(aq) + 2e⁻ → Cu(s) ───────────────────────────── Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s)

Unequal electrons — scale up first

If the two half-equations have different numbers of electrons, multiply each equation by whatever factor makes the electron counts match (usually the lowest common multiple) before adding and cancelling.

5Fe²⁺(aq) → 5Fe³⁺(aq) + 5e⁻ (×5) MnO4⁻(aq) + 8H⁺(aq) + 5e⁻ → Mn²⁺(aq) + 4H₂O(l) ───────────────────────────────────────────── 5Fe²⁺(aq) + MnO4⁻(aq) + 8H⁺(aq) → 5Fe³⁺(aq) + Mn²⁺(aq) + 4H₂O(l)

Full method for combining half-equations

  1. Write both ionic half-equations (or identify which elements change oxidation number, and by how much).
  2. Find the lowest common multiple of the electron numbers in each half-equation.
  3. Multiply each half-equation so both have that same number of electrons.
  4. Add the two half-equations together and cancel the electrons.
  5. Balance any remaining H and O atoms (usually with H⁺ and H₂O, in acidic conditions).
  6. Check the equation balances for charge on both sides — total charge on the left must equal total charge on the right.
Step 6 (checking charge balance) is the step most students skip — but it's a fast, reliable way to catch an error before you submit an answer. If the charges don't match, you've made a mistake somewhere.

Worked example — using oxidation number change ratios directly

For Fe²⁺ + ClO₃⁻ → Fe³⁺ + Cl⁻ (in acid): Fe changes +2→+3 (a 1-electron change); Cl changes +5→−1 (a 6-electron change). Ratio of Fe : ClO₃⁻ = 6 : 1.

6Fe²⁺ + ClO₃⁻ + 6H⁺ → 6Fe³⁺ + Cl⁻ + 3H₂O

Charge check: LHS = (6×+2) + (−1) + (6×+1) = +17. RHS = (6×+3) + (−1) = +17. Balanced ✓

Method
What do you do first if two half-equations have a different number of electrons?
Multiply each half-equation by whatever factor is needed so both have the same (lowest common multiple) number of electrons, before adding them together.
Method
After adding half-equations and cancelling electrons, what should you always check?
That the total charge on the left-hand side equals the total charge on the right-hand side of the final equation.
Apply
Combine Zn(s) → Zn²⁺(aq) + 2e⁻ and Fe³⁺(aq) + e⁻ → Fe²⁺(aq) into one full equation.
Multiply the iron equation by 2: 2Fe³⁺(aq) + 2e⁻ → 2Fe²⁺(aq). Add: Zn(s) + 2Fe³⁺(aq) → Zn²⁺(aq) + 2Fe²⁺(aq).
Balance
In acidic conditions, what two species are usually used to balance leftover H and O atoms?
H⁺ ions and H₂O molecules.
