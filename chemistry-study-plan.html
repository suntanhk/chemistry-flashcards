4
Summer 2 Supplementary Notes (not in Book 1)
▾

These three areas need to be taught/revised from other resources since Book 1 doesn't include them. Outline below to guide what to look for.

NMR Spectroscopy

Chemical shift (δ, ppm) and reference to TMS; splitting patterns via the n+1 rule; integration ratio = relative proton count; use of D₂O shake to identify O–H/N–H exchangeable protons; distinguishing environments in ¹³C NMR.

Chromatography

TLC and paper chromatography set-up; Rf value calculation (distance run by spot ÷ distance run by solvent front); using Rf and co-spotting to confirm/rule out identity; assessing purity from number of spots.

Chirality & Optical Isomerism

Chiral centre = carbon with four different groups attached; enantiomers rotate plane-polarised light in opposite directions; racemic mixture = no net rotation; link to E/Z geometric isomerism already met in Topic 6.1.

Recommend sourcing: school's own Summer 2 class notes, Book 2 (if accessible), or Pearson ActiveLearn — the school's Chemistry department already provides comprehensive class notes per the curriculum map, which should cover this.
